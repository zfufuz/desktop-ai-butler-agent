# 桌面 AI 管家 产品计划

## 项目定位

一个运行在 Windows 桌面的个人 AI 管家，具备动态立绘、聊天、语音回复、文件操作、代码辅助、表格处理、任务规划和使用时长提醒能力。

## MVP 功能

- 动态立绘
- 文本聊天
- 语音回复
- 文件读取、写入、修改、删除
- 删除文件前必须确认
- 表格创建和编辑
- 代码读取、解释和修改
- 修改代码前生成 diff 或备份
- 简单任务规划
- 使用电脑时间过长提醒
- 操作日志
- 基础设置页

## 第一版不做

- RAG 知识库
- MCP
- 多 Agent
- 插件市场
- 自动控制鼠标键盘
- 自动发消息或付款
